Metatag: Dublin Core
--------------------
This module adds the fifteen Dublin Core Metadata Element Set [1] (plus one) to
the available meta tags, as defined by the Dublin Core Metadata Institute [2].

The following tags are provided:
* dcterms.contributor
* dcterms.coverage
* dcterms.creator
* dcterms.date
* dcterms.description
* dcterms.format
* dcterms.identifier
* dcterms.language
* dcterms.modified (additional, not part of the DCES)
* dcterms.publisher
* dcterms.relation
* dcterms.rights
* dcterms.source
* dcterms.subject
* dcterms.title
* dcterms.type
